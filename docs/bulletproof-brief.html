Bulletproof Partnership Brief

Overview

Bonito is a unified AI control plane. One platform to connect, govern, route, and manage AI workloads across cloud providers — AWS, Azure, Google, OpenAI, Anthropic, and more. It provides organizations with a single API endpoint, unified cost tracking, compliance controls, and a built-in framework for deploying autonomous AI agents.

This document outlines three partnership models between Bonito AI and Bulletproof — each designed to reduce operational costs, create new revenue streams, and position Bulletproof as an AI-forward managed services provider.

Integrating Your Engineering Team

Bulletproof's engineering team currently uses Claude Code. Routing all AI requests through Bonito requires two environment variables:

# Point Claude Code to Bonito gateway
ANTHROPIC_BASE_URL=https://api.getbonito.com/v1
ANTHROPIC_API_KEY=bn-your-key-here

Every request then routes through Bonito with the following capabilities:

  • Cost tracking by engineer, by model, by day
  • Budget caps per project so no team blows through credits
  • Model policies — restrict which models can be used for specific client work
  • Full audit trail of every AI request across the organization
  • Intelligent failover — if one provider goes down, Bonito automatically reroutes to an equivalent model on another provider

For Bulletproof's compliance-sensitive clients: Every AI interaction is logged with timestamps, user identity, model used, tokens consumed, and cost. This audit trail is available via dashboard and API export — ready for SOC 2, gaming commission, and municipal audit requirements.

Bulletproof Partnership Brief
1 Tier 1 Support Automation

Bulletproof manages Tier 1 support for 50,000+ end users across 300+ client environments. Bonito's agentic framework can power AI agents that act as the first line of response — resolving common tickets automatically and escalating complex issues with full context to human agents.

How It Works

  • Bulletproof provides client runbooks, SOPs, and knowledge base articles
  • Bonito ingests them into isolated Knowledge Bases — per client or client group
  • AI agents are configured per environment with the right documentation, approved models, and escalation rules
  • Agents integrate into existing ticketing workflows via API or embeddable widget
  • Every interaction is logged with a full audit trail

What Bulletproof Retains Control Of

  • Which AI models the agents can use
  • Budget caps per agent, per client environment
  • Human-in-the-loop approval gates on sensitive actions (account changes, access provisioning)
  • Complete visibility into agent performance, accuracy, and cost

2 White-Label Managed AI

Bulletproof currently offers managed IT and managed security. Bonito can be white-labeled under Bulletproof's brand as a third pillar: managed AI.

Bulletproof's clients receive governed AI access — gateway, agents, knowledge base, and compliance — without building anything themselves. Bulletproof manages it as an extension of their existing service offering with their own pricing and margins.

  • Bonito platform with Bulletproof branding, colors, and custom domain
  • Multi-tenant architecture — each client gets a fully isolated environment
  • Bulletproof acts as the admin layer, managing client organizations and setting policies
  • Engineering team builds and configures agents for clients using familiar tools

Strategic value: Gaming and government clients cannot adopt AI without proper governance and audit controls. They need a governed, auditable platform from a vendor they already trust. Bulletproof becomes a one-stop shop: managed IT + managed security + managed AI.

3 Referral Partnership

When Bulletproof's clients inquire about AI strategy or onboarding, Bulletproof refers them to Bonito. Bonito handles the sale, onboarding, and ongoing support. Bulletproof earns a referral fee on every closed deal.

  • Zero engineering effort from Bulletproof
  • Monetizes conversations already happening with existing clients
  • Government and gaming clients referred by a trusted MSP convert at higher rates
  • Builds naturally toward the white-label model over time

Recommended Path Forward

We recommend starting with Scenario 1 (Tier 1 support agents) as an immediate proof of value, with Scenario 3 (referral) running in parallel at zero cost. Scenario 2 (white-label) becomes the long-term play once the agent deployment demonstrates ROI.

  • Sample runbooks or SOPs from 2–3 client environments for a pilot build
  • A brief walkthrough of the current ticketing workflow to plan the integration point
  • Any compliance requirements specific to government or gaming clients

Bonito will build a functional pilot agent that Bulletproof can interact with and evaluate before any commercial commitment.
